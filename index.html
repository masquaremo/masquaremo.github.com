Github+Octopressデプロイ

デプロイ前の環境

  • Ruby Installerの Ruby 1.9.2-p290 インストール済み

手順

  1. 上記のサイトから、DevKitをダウンロード→解凍

    ruby dk.rb init ruby dk.rb review ruby dk.rb install

  2. リポジトリをつくる。

  3. Octopressをインストール
  4. ブログの設定
  5. 記事の投稿

関連情報

Windows環境で _config.ymlファイルや記事ファイルに日本語が含まれていて rake gen_deploy で

invalid byte sequence in Windows-31J

とエラーになる場合、環境変数 LANGを設定する。

LANG=ja_JP.utf8

みたいに。
